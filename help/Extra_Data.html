Extra Data Editor

      Purpose: The Extra Data Editor adds extra chunks of Data to the map file. What this is good for is things like adding the monster

     percentage chance for the current area or possibly other information about the world.

 

  1. View Data – This option shows all the extra Chunks of data that you have added
  2. Edit Data – This option allows you to select one of the already added data chunks and change its value
  3. Add Data – This option adds an extra set of Data to the map file and lets you set if with a value
  4. Delete Data – This option deletes the last added piece of data.
