Magus Map Maker

Overview

The purpose of the Magus Map Maker is to generate maps for use in any application that my need tile maps. The program is capable of writing a map file containing 3 layers as well as one tile attribute layer. The Map Maker is also couple with another tile editing program called the Tilenator. The Tilenator can be used to load a tilesheet and then to break it into its component tiles. It can also be used to take a directory of tiles and create a tilesheet.
