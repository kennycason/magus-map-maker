Load Tiles

These options allows you to load tiles of any dimension and virtually any file type. The supported file types are BMP(*.bmp), JPEG(*.jpg|*.jpeg), PNG(*.png), TIFF(*.tif|*.tiff), TGA(*.tga). However, there are some guidelines and it should be noted that Map Maker bases all its information off of the first tile it loads.

Filenames must by <integer>.<file type>, where integer is > 0 and less than 256, and file type is one of the supported file types.
